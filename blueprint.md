# AgroTech Website Blueprint

## Overview

This document outlines the structure, features, and development plan for the AgroTech website. The website serves as an educational platform ("Edutech") about the agribusiness sector, covering topics from technology and sustainability to its economic impact.

## Implemented Features & Design

### Standardized Navigation Menu

A consistent, modern, and responsive navigation menu has been implemented across all pages of the website.

*   **Structure:** The menu is a hamburger-style menu that expands into a full-screen overlay.
*   **Branding:** It includes the "AgroTech" logo.
*   **Links:** The menu provides navigation to the following pages:
    *   HOME (`index.html`)
    *   TECNOLOGIA NO CAMPO (`tecnologiaNoCampo.html`)
    *   CONTRIBUIÇÃO AO PIB (`contribuicaoAoPib.html`)
    *   LOGÍSTICA E EXPORTAÇÃO (`logisticaExportacao.html`)
    *   SUSTENTABILIDADE (`sustentabilidade.html`)
    *   COMO FUNCIONA ? (`comoFunciona.html`)
    *   CASOS DE SUCESSO DO AGRONEGÓCIO (`casosDeSucesso.html`)
    *   FEIRAS E EVENTOS (`feirasEeventos.html`)
    *   PRATICANDO (`jogo1.html`)
    *   SOBRE NÓS (`sobreNos.html`)
*   **"Precisa de Ajuda?" Modal:**
    *   A "PRECISA DE AJUDA?" button opens a modal window with a contact form.
    *   The form collects the user's email and a message.
    *   Upon submission, a confirmation modal is displayed.
